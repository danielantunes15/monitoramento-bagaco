# FireGuard - Sistema de Monitoramento de Temperatura

Sistema moderno para monitoramento em tempo real de temperatura e detecção de princípios de incêndio em pilhas de bagaço.

## Funcionalidades

- **Monitoramento em tempo real**: Atualização contínua da temperatura
- **Alertas inteligentes**: Notificações visuais e sonoras para temperaturas críticas
- **Visualização gráfica**: Histórico de temperatura com gráficos interativos
- **Multi-sensor**: Monitoramento de até 4 sensores simultaneamente
- **Interface responsiva**: Funciona em desktop e dispositivos móveis
- **Exportação de dados**: Capacidade de exportar dados para análise

## Tecnologias Utilizadas

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js para gráficos
- Node.js com Express para o servidor
- WebSocket para comunicação em tempo real
- Design moderno com Flexbox/Grid CSS

## Instalação e Execução

### Pré-requisitos
- Node.js (v14 ou superior)
- NPM (gerenciador de pacotes)

### Passos para instalação

1. Clone o repositório ou extraia os arquivos
2. Navegue até a pasta do servidor:# monitoramento-bagaco
